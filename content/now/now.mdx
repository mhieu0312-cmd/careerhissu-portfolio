## 🎯 Mục tiêu lớn trong tuần này
Mặc dù GPA năm nhất có hơi thoi thóp một chút, nhưng t không được phép nản lòng. Tuần này t tập trung toàn lực vào hai thứ quan trọng nhất: **Kiến trúc Web thực tế** và **Cơ bản Lập trình hướng đối tượng (OOP)**.

### 💻 Về mặt Lập trình & Kỹ thuật
- **Setup Portfolio / Second Brain**:
  - Đang phối hợp cùng con AI **Sil** để tự xây dựng bộ khung Next.js + Tailwind CSS từ vạch xuất phát.
  - Tự tay viết code xử lý routing, cấu trúc layout và tự động chuyển đổi file **MDX** thành HTML thay vì lười biếng dùng template dựng sẵn.
  - Tối ưu hóa UI/UX với cả hai chế độ Sáng/Tối (Light/Dark mode) và hiệu ứng chuyển động mượt mà.
- **Học OOP C++**:
  - Ôn tập kỹ các khái niệm cơ bản của hướng đối tượng: Đóng gói (Encapsulation), Kế thừa (Inheritance), Đa hình (Polymorphism) để chuẩn bị cho kỳ học tiếp theo.
  - Code thử nghiệm một vài chương trình Console quản lý thư viện đơn giản.

### 🏃‍♂️ Về mặt Phát triển Cá nhân
- **Chạy bộ**: Đặt mục tiêu chạy tối thiểu 3km mỗi chiều tối để giải tỏa đầu óc và duy trì sức bền.
- **Giấc ngủ**: Cố gắng tắt máy tính trước 23:30 và chìm vào giấc ngủ trước 00:00. Không được thức đêm cày code vô tội vạ nữa!
